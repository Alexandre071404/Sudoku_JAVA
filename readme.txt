Jeux de sudoku.

Petit jeu de sudoku avec plusieurs niveaux de difficultés avec un modèle de POO JAVA MVC le controller n'ayant pas de dossier ni de classe attitrée il contenu dans la View.

Un axe d'amélioration serait d'améliorer la lisibilé et la solidité du code en créant une classe Controller.

Arborescence :

Model qui contient la grille de sudoku.

View qui contient les classes permettant l'affichage des différents éléments du jeu ainsi que le controller qui modifie le modèle.

Pour lancer le projet : dans le terminal lancer la commande 'make run'

Mise à disposition de l'archive du jeu.

Ps: De petits problèmes peuvent subvenir à cause des déclarations d'importation dans ces cas là une solution peut éviter cela: enlever les 'src.' lors de la déclaration d'importation des classes.
Si les problèmes persistent revoir les import du projet.


Sudoku Game

A small Sudoku game with various difficulty levels, developed using a Java OOP MVC model. The controller doesn't have its own folder or class; it is contained within the View.

One potential improvement would be to enhance the readability and robustness of the code by creating a dedicated Controller class.

Structure:

Model contains the Sudoku grid.

View includes the classes responsible for displaying the various elements of the game, as well as the controller that modifies the model.

To launch the project: run the command 'make run' in the terminal.

The game archive is provided.

Note: Small issues may arise due to import declarations. In such cases, one possible solution is to remove the 'src.' prefix when declaring class imports.
If problems persist, review the project imports.
